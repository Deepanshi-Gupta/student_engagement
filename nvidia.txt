Based on the page you are viewing, **this is the Nemotron-3-Ultra-550B-A55B model by NVIDIA**, hosted on the NVIDIA NIM (NVIDIA Inference Microservice) platform.

It is a massive, cutting-edge hybrid **Mamba-Transformer Mixture of Experts (MoE)** large language model with a 1-million-token context window. It is specifically designed for complex, agentic reasoning, long-document processing, coding, and multi-step planning tasks.

## How Can You Use It?

Because the hosted endpoint is OpenAI-compliant, you can easily integrate it into your projects using standard LLM libraries. Here is exactly how to set it up:

### 1. Python Integration

You can use the standard `openai` SDK to call the model. Ensure you have your API key set up as an environment variable (`$NVIDIA_API_KEY`).

```python
from openai import OpenAI

client = OpenAI(
    base_url="https://integrate.api.nvidia.com/v1",
    api_key="$NVIDIA_API_KEY"
)

completion = client.chat.completions.create(
    model="nvidia/nemotron-3-ultra-550b-a55b",
    messages=[
        {"role": "user", "content": "Your prompt goes here"}
    ],
    temperature=1,
    top_p=0.95,
    max_tokens=16384,
    extra_body={
        "chat_template_kwargs": {"enable_thinking": True},
        "reasoning_budget": 16384
    },
    stream=True
)

for chunk in completion:
    if not chunk.choices:
        continue
    
    # Extract deep reasoning tokens if available
    reasoning = getattr(chunk.choices[0].delta, "reasoning_content", None)
    if reasoning:
        print(reasoning, end="")
        
    # Extract final text output
    if chunk.choices[0].delta.content is not None:
        print(chunk.choices[0].delta.content, end="")

```
